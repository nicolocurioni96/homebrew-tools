# homebrew-tools

Homebrew tap for [Shotkit](https://github.com/nicolocurioni96/withnico-skills) and other withnico tools.

## Install

```bash
brew tap nicolocurioni96/tools
brew install shotkit
```

## Formulae

| Formula | Description |
|---------|-------------|
| `shotkit` | App Store screenshot pipeline: auto-capture, composite, and validate |
